TABLA DE DEFINICION DE ETIQUETAS HTML
etiqueta DEFINICION SINTAXIS
HTML Es una etiqueta de html capaz de crear una página con código html
a Es una etiqueta html que se utiliza para añadir enlaces a una página de html
body Es una etiqueta html que se utiliza para la estructura o cuerpo de un texto en la página html
head Es una etiqueta html que se utiliza para el encabezado de una página web con html
div Es una etiqueta html que se utiliza para crear secciones o agrupar contenidos
strong Es una etiqueta html que se utiliza para marcar partes especificas de un texto
br Es una etiqueta html que se utliza para realizar saltos entre lineas, dejando asi espacios entre párrafos
h1-h6 Es una etiqueta html que se utiliza para dar 6 tamaños distintos en las letras de una página html
img Es una etiqueta html que permite añadir una imagen a una página html ya sea desde un enlace o un archivo
ol Es una etiqueta html que se utiliza para listas ordenadas ya sea numerica o alfabeticamente
ul Es una etiqueta html que se utiliza para crear listas desordenadas
li Es una etiqueta html que se utliza para representar un elemento en una lista y ese elemento debe estar contenido en un elemento que sea una (ol),(ul),(menu)
p Es una etiqueta html que representa un párrafo en una página web de html
span Es una etiqueta de html que se utiliza para aplicar estilos a un texto o agrupar elementos en línea
b El propósito de esta etiqueta html es llamar la atención del lector al texto que contiene sin ninguna otra implicación.
title Esta etiqueta define el título de la página que aparece en la pestaña del navegador
table Esta etiqueta html define una tabla en la cual se pueden agregar datos
tr Esta etiqueta de html define una fila en una tabla (table)
td Define una celda en una tabla
th Esta etiqueta html representa una celda de encabezado en una tabla (table)
form Esta etiqueta html define un formulario para recopilar información de algun usuario
input Esta etiqueta html define un campo de entrada para un usuario, como una casilla de verificación, un botón de radio o un campo de texto
button Esta etiqueta de html define un botón en el que un usuario pueda hacer clic
